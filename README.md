1. Setup the project
   ```bash
   npm install
   npm run dev
   ```
2. Environment configuration
   - Copy the `.env.example` to a new file named `.env` and adjust as needed.
   ```bash
   cp .env.example .env
   ```

3. Running the application
   - Frontend App (Nuxt): `http://localhost:3000`
   - Backend Admin (Directus): `http://localhost:8055`
